# slogar

Kotlin-библиотека для разбиения русских слов на слоги.

## Школа разбиения

`slogar` по умолчанию использует школьно-фонетическое разбиение русских слов на слоги.

Это не алгоритм переноса строк. Переносы учитывают орфографические ограничения, морфемы и типографские правила. `slogar` делит слово на произносительные слоги.

Базовые принципы:

- слог строится вокруг гласной;
- количество слогов равно количеству гласных;
- `й`, `ь`, `ъ` не образуют отдельный слог;
- `ё` считается самостоятельной гласной;
- исходный регистр сохраняется в результате;
- слова без гласных возвращаются одним фрагментом с предупреждением.

## Слои правил

Правила применяются к участку между двумя соседними гласными.

1. Стык гласных: граница проходит между гласными.
   - `аист` -> `а-ист`
   - `поэт` -> `по-эт`
2. Разделительные `ь` и `ъ`: знак остается в предыдущем слоге.
   - `семья` -> `семь-я`
   - `подъезд` -> `подъ-езд`
3. `й` после гласной: `й` остается в предыдущем слоге.
   - `майка` -> `май-ка`
   - `район` -> `рай-он`
4. Одна согласная между гласными: согласная отходит к следующему слогу.
   - `коса` -> `ко-са`
5. Группа согласных: по базовому правилу последняя согласная отходит к следующему слогу.
   - `кошка` -> `кош-ка`
   - `сестра` -> `сест-ра`
